# WEPSEED 用户协议

最后更新：2026-07-16

欢迎使用 WEPSEED（下称「本应用」）。本应用为开源、本地优先的 RSS 阅读工具。使用本应用即表示你同意本协议。若不同意，请停止使用。

本应用源码以 [MIT License](../LICENSE) 发布。你可以在遵守 MIT 条款的前提下使用、修改与再分发软件本身。本协议约束的是**你使用本应用的方式**，不替代 MIT。

## 2. 本地优先与责任

- 订阅源、阅读记录、收藏、设置、评论缓存等默认保存在**你的设备本地**。
- 你应对自己添加的订阅源、笔记、导入的 OPML 与生成的内容负责。
- 请勿将本应用用于任何违法、侵权或骚扰用途。

## 3. 大语言模型（LLM）

- 评论等 LLM 能力仅在你**自行配置** API 提供商与密钥后启用。
- 请求发往**你指定的**第三方服务端点；本应用开发者不运营你的模型账号，也不对第三方服务可用性、计费与内容负责。
- 你应遵守所使用 LLM 服务商的服务条款与当地法律法规。
