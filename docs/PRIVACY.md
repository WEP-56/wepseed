# WEPSEED 隐私政策

最后更新：2026-07-16

WEPSEED 是**本地优先**的 RSS 阅读应用。我们（本仓库维护者）**不运营账号体系**，也**不主动收集**你的阅读数据到自有服务器。

## 1. 本机存储的数据

可能保存在你设备上的信息包括（视功能使用而定）：

- RSS 订阅源列表与文章缓存（标题、摘要、正文 HTML、已读/收藏状态等）
- 应用设置（主题、字号、刷新频率、通知开关等）
- 网友人设、LLM 提供商配置（不含密钥明文，见下）
- 由 LLM 生成的评论缓存
- 阅读温度相关事件（dwell / binge 等本地时间轴）

卸载应用或清除应用数据将删除上述本地数据（操作系统行为为准）。

## 2. API 密钥

- LLM API Key 使用系统级安全存储（如 Android Keystore 相关机制，经 `flutter_secure_storage`）。
- Key **不会**上传到 WEPSEED 开发者服务器（本项目无此类服务）。

## 3. 网络请求

在你使用相应功能时，本应用可能发起：

| 目的 | 目标 | 说明 |
|------|------|------|
| 拉取订阅 | 你添加的 RSS/Atom URL | 由你选择的源决定 |
| LLM 评论 | 你配置的 API Base URL | 仅在启用评论且配置有效时 |
| 检查更新 | `api.github.com`（本仓库 Releases） | 仅在你点击「检查更新」等主动操作时 |
| 打开链接 | 系统浏览器 | 原文、协议、GitHub 等外链 |

本应用**不会**把你的订阅列表或阅读全文发送到开发者后端（不存在该后端）。

## 4. 通知

若你开启「更新通知」，系统本地通知可能展示新文章标题与源名称；通知在本地生成，不经 WEPSEED 推送服务器。

## 5. 第三方

- **GitHub**：检查更新 / 下载 APK 时访问 GitHub 公开 API 与文件 CDN，受 [GitHub 隐私政策](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) 约束。
- **LLM 服务商**：你自行选择的提供商将接收你发起的提示与文章相关片段，受该服务商政策约束。
- **订阅源站点**：RSS 请求受各站点政策约束。

## 6. 儿童

本应用不面向 13 岁以下儿童专门设计；请监护人指导下使用。

## 7. 变更

本政策更新将发布于本文件。重大变更可通过 GitHub Release 说明提示。
